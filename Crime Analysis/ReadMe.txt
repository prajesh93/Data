1.	Store the dataset in a particular directory
2.	Import the dataset into SQL Server Management Studio and then run the file SQLcleaning.sql.
3.	Open the file datacleaning.R.
4.	Set the working directory which is the directory where the database is stored.
5.	Run the file datacleaning.R. It will clean the dataset.
6.	Once the dataset is cleaned, run the visualization.R file. It will show the various visualizations which we have created.
7.	Finally, run the file crimeanalysis.R which will train the model for the Neural Network, Bayesian Classification, Support Vector Machine and Decision Tree. It will then test it against the testing data.
8.	The accuracy and confusion matrix will be shown in the console.
